# psv

psv is a lightweight desktop app for quickly opening and reading text and log files, including very large ones that would make other editors slow to start or scroll. It opens files almost instantly, lets you jump straight to a line or search for text, and figures out the right character encoding on its own so files display correctly without any setup.

It's built for keeping an eye on logs and other text as they change: turn on live tail and the view follows new lines as they're written to the file, much like watching a log scroll by in a terminal. You can also select and copy text, adjust fonts and colors to your liking, and switch between light and dark appearance. psv runs on Windows, Linux, and macOS.
